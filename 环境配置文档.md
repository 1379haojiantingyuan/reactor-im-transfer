# 环境配置文档

## 1. 操作系统
- **推荐**: Linux (Ubuntu 20.04 LTS 或更高版本)
- **内核**: 2.6.32+ (支持 Epoll)

## 2. 编译工具链
需安装 C++ 编译器和构建工具：
- **GCC/G++**: 支持 C++17 标准 (建议 9.0+)
- **Make**: GNU Make 4.0+

## 3. 依赖库
- **Pthread**: POSIX 线程库 (通常随 libc 安装)
- **Ncurses**: 用于客户端终端界面 (`libncurses-dev`)

## 4. 环境搭建命令 (Ubuntu/Debian)

```bash
# 更新软件源
sudo apt-get update

# 安装编译必需包 (包含 g++, make, pthread 等)
sudo apt-get install -y build-essential

# 安装 Ncurses 开发库
sudo apt-get install -y libncurses-dev
```

## 5. 验证环境

安装完成后，可运行以下命令检查版本：

```bash
g++ --version
make --version
```

编译测试：

```bash
# 在项目根目录
make clean
make
```
